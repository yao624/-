2026-03-25 创建 Meta 文案库相关表
- 迁移脚本: database/migrations/2026_03_25_100000_create_meta_copy_library_tables.php
- SQL文件（LZC）: backend/sql/lzc/tool_copy_library.sql
- 涉及表:
  - meta_copy_libraries: 文案库根表，区分个人/企业文案库、所属用户、可见范围、状态等
  - meta_copy_folders: 文案库多级文件夹，支持最多 20 级、统计直系及合计文案条数
  - meta_copy_items: 单条文案内容（primary_text/headline/description/备注 等），关联库和文件夹
  - meta_copy_library_permissions: 文案库授权表，按用户/角色控制可管理、可写、可删除权限
  - meta_copy_performance_daily: 文案维度投放日快照，记录展示/点击/花费/转化/收入等指标
- 用途说明: 对齐 XMP 的“文案库”能力，在投放侧提供个人/企业级文案管理、分组、多级文件夹和权限控制，并可按文案汇总投放表现数据，为选文案、复用和复盘提供基础数据结构。

2026-03-27 创建 Meta 报表看板相关表

- SQL文件（LZC）: backend/sql/lzc/tool_report_dashboard.sql

  - meta_report_dashboard_folders: 报表看板左侧文件夹树，多级目录 + 排序 + 状态
  - meta_report_dashboards: 看板主表，包含名称、所属文件夹、位置（我的/分享/系统）、渠道、类型、默认筛选、上次保存时间等

- 用途说明: 支撑前端“报表看板/自定义看板”功能，提供按用户组织的看板目录、看板基础信息、卡片布局与查询配置，以及看板共享/协作权限的持久化结构。
